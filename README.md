# Simple Cash API

Простой пример API на Go с Redis-кешированием и ограничением запросов.

## Что делает

- кэширует погодные данные по городу на 30 секунд
- ограничивает пользователя до 5 запросов в минуту
- сохраняет последние запросы пользователя
- собирает топ популярных городов
- логирует запросы в JSON формате

## Endpoints

- `GET /weather/:city` — получить погоду по городу
- `GET /user/history` — список последних 5 запросов пользователя
- `GET /top-cities` — топ популярных городов

## Запуск

1. Перейдите в папку проекта:
   ```bash
   cd simple_cash_api
   ```

2. Установите зависимости:
   ```bash
   go mod tidy
   ```

3. Запустите сервис:
   ```bash
   go run cmd/app.go
   ```

4. Или через Docker Compose:
   ```bash
   docker compose up --build
   ```

## Требования

- Go 1.26+
- Redis (docker-compose уже включает `redis:7-alpine`)

## Примечания

Запросы к `/weather/:city` должны содержать заголовок `User-Id`.
